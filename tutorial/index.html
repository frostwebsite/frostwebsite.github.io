  • Single m3u8 player - Watch IPTV online from an m3u8 link.

Usage

IPTVnator Web

  1. Copy the URL
  2. Click on the plus button at the top right
  3. Click on “Add via URL”
  4. Paste the URL
  5. Click on add playlist

VLC Windows

  1. Copy the URL
  2. Press CTRL + V and then press enter.

VLC macOS

  1. Copy the URL
  2. Press Command + V and then press enter.

VLC Android

  1. Copy the URL
  2. Go to the “More” tab at the bottom
  3. Press on new stream
  4. Click on the play icon

VLC iPhone

  1. Copy the URL
  2. Go to the “Network” tab at the bottom
  3. Press on new stream
  4. Click on the play icon

PotPlayer (Windows)

  1. Open PotPlayer
  2. Copy the URL
  3. Ctrl+V

Kodi (Google TV, FireStick, Nvidia Shield, Google TV)
  1. Install from Repository → Kodi Add-on repository
  2. PVR clients → look for IPTV Simple Client and install
  3. Configure → 3rd option
  4. Set Location to Remote path
  5. Paste the URL
  6. Make sure it's enabled and click OK. Wait for the channels to load in and enjoy!
